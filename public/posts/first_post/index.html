Apache Airflow

    Apache Airflow 是一个使用 Python 语言编写的 Data Pipeline 调度和监控工作流的开源平台，可以用于编排复杂工作流。它是通过 DAG（Directed acyclic graph 有向无环图）来管理任务流程的任务调度工具——支持容错，设置任务依赖关系和时间调度，处理失败任务，生成失败的任务报告和警报。
